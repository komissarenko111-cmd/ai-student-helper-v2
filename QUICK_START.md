## 🎯 Как использовать

### Базовое использование
1. Введите вопрос в текстовое поле
2. Нажмите "Отправить" или Shift+Enter
3. Получите ответ от AI

### Изменение параметров
- **Модель** - выберите в боковой панели
- **Креативность** - регулируйте температуру (0-2)

### Очистка истории
- Нажмите кнопку "Очистить историю" в боковой панели

## 🐛 Часто встречающиеся проблемы

**P: "ModuleNotFoundError: No module named 'streamlit'"**
О: Запустите `pip install -r requirements.txt`

**P: "API key is invalid"**
О: Проверьте, что ключ скопирован полностью

**P: Сообщения не сохраняются между перезагрузками**
О: Это нормально для Streamlit (базовая функция). Для постоянного хранения используйте БД.

**P: Приложение медленное**
О: Ограничьте контекст до 10-20 сообщений (см. INTEGRATION_GUIDE.md)

## 🚀 Запуск в облаке (Streamlit Cloud)

### Бесплатный хостинг на Streamlit Cloud

1. Загрузьте код на GitHub
2. Создайте `requirements.txt` (уже есть ✓)
3. Перейдите https://streamlit.io/cloud
4. Нажмите "New app"
5. Выберите ваш репозиторий
6. В Settings добавьте секрет:
   ```
   GROQ_API_KEY = gsk_...
   ```
